Project Journey

From challenge to implemented solution

Start
01 — Challenge

The Business Problem

The client needed to visualize in real-time all the checklists performed in their transport operation. These checklists evaluate vehicle safety items before trips.

  • Immediate approval/rejection analysis
  • Identify reasons for rejection
  • Evaluate operator performance
  • Track daily trends
Planning
02 — Solution

Solution Architecture

A complete Power BI view was built using data processed in SQL Server.

  • Overall Approval Rate
  • Approved vs Rejected Checklists
  • Average Execution Time
  • Operator Ranking
  • Hourly Time Series with Dynamic UTC
Development
03 — Ingestion

Data Processing

Creation of a mirrored database in SQL Server with optimized views.

  • Duration calculations
  • Status categorization
  • Dynamic extraction of rejected items
  • Query optimization for performance
Implementation
04 — Visualization

Consumption Layer

DirectQuery connection for real-time data.

  • Handling of rejected items table
  • DAX measures development
  • Time table with UTCNOW()
  • Dynamic and interactive KPIs
Result
05 — Impact

Measurable Success

The final dashboard allowed real-time monitoring of the operation, facilitating quick decisions and increasing control over recurring vehicle failures.

  • Complete data visibility
  • Transformed risk management
  • Data-driven decisions
  • Reduction of operational failures
