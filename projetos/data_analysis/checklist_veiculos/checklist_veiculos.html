Do desafio à solução implementada

Início
01 — Desafio

O Problema de Negócio

O Cliente precisava visualizar em tempo real todos os checklists realizados em sua operação de transporte. Esses checklists avaliam itens de segurança dos veículos antes das viagens.

  • Análise imediata de aprovação/reprovação
  • Identificar motivos de reprova
  • Avaliar desempenho dos operadores
  • Acompanhar tendências diárias
Planejamento
02 — Solução

Arquitetura da Solução

Foi construída uma visão completa em Power BI utilizando dados tratados no SQL Server.

  • Taxa de Aprovação Geral
  • Checklists Aprovados x Reprovados
  • Tempo médio de execução
  • Ranking dos operadores
  • Série temporal hora a hora com UTC dinâmica
Desenvolvimento
03 — Ingestão

Tratamento de Dados

Criação de base espelhada no SQL Server com views otimizadas.

  • Cálculos de duração
  • Categorização de status
  • Extração dinâmica de itens reprovados
  • Otimização de queries para performance
Implementação
04 — Visualização

Camada de Consumo

Conexão via DirectQuery para dados em tempo real.

  • Tratamento de tabela de itens de reprovação
  • Desenvolvimento de medidas DAX
  • Tabela de Hora com UTCNOW()
  • KPIs dinâmicos e interativos
Resultado
05 — Impacto

Sucesso Mensurável

O dashboard final permitiu acompanhamento em tempo real da operação, facilitando decisões rápidas e aumentando o controle sobre falhas recorrentes nos veículos.

  • Visibilidade completa dos dados
  • Gestão de risco transformada
  • Decisões baseadas em dados
  • Redução de falhas operacionais
