Monitoramento de Checklists em Tempo Real – BASF Agro

Problema de Negócio

A BASF Agro precisava visualizar em tempo real todos os checklists realizados em sua operação de transporte. Esses checklists avaliam itens mecânicos e de segurança dos veículos antes das viagens, influenciando diretamente a eficiência e o gerenciamento de risco da operação.

Era fundamental permitir uma análise imediata de aprovação/reprovação, identificar os principais motivos de reprova, avaliar desempenho dos operadores e acompanhar tendências ao longo do dia.

Solução Desenvolvida

Foi construída uma visão completa em Power BI utilizando dados tratados no SQL Server. Os principais indicadores foram:

  • ✔ Taxa de Aprovação Geral
  • ✔ Checklists Aprovados x Reprovados
  • ✔ Tempo médio de execução
  • ✔ Ranking dos operadores
  • ✔ Série temporal hora a hora
  • ✔ Itens mais recorrentes de reprovação

Etapa 1 – Tratamento no SQL Server

Foi criada uma base espelhada chamada POWERBI. A view abaixo organiza e padroniza os dados:

USE [POWERBI] GO ALTER VIEW [dbo].[CHECKLISTS] AS ( SELECT [... código completo aqui ...] ) GO

Etapa 2 – Modelagem no Power BI

Foi criada uma consulta no Power Query para listar todos os itens de reprovação:

let Fonte = CHECKLISTS, LinhasNaoVazias = Table.SelectRows(Fonte, each ([ITENS_REPROVA] <> null and [ITENS_REPROVA] <> "")), ColunaListas = Table.AddColumn(LinhasNaoVazias, "ListaItens", each Text.Split([ITENS_REPROVA], ";")), ExpandirItens = Table.ExpandListColumn(ColunaListas, "ListaItens"), ItensTrim = Table.TransformColumns(ExpandirItens, {{"ListaItens", Text.Trim, type text}}), SelecionarItem = Table.SelectColumns(ItensTrim, {"ListaItens"}), ItensDistinct = Table.Distinct(SelecionarItem, {"ListaItens"}), ItensReprova = Table.RenameColumns(ItensDistinct, {{"ListaItens", "ITEM"}}) in ItensReprova

Resultado Final

O dashboard final permitiu acompanhamento em tempo real da operação, facilitando decisões rápidas e aumentando o controle sobre falhas recorrentes nos veículos.
