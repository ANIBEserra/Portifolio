01 — DESAFIO

O Problema de Negócio

A BASF Agro precisava visualizar em tempo real todos os checklists realizados em sua operação de transporte. Esses checklists avaliam itens mecânicos e de segurança dos veículos antes das viagens, influenciando diretamente a eficiência e o gerenciamento de risco da operação.

Era fundamental permitir uma análise imediata de aprovação/reprovação, identificar os principais motivos de reprova, avaliar desempenho dos operadores e acompanhar tendências ao longo do dia.

02 — SOLUÇÃO

Arquitetura da Solução

Foi construída uma visão completa em Power BI utilizando dados tratados no SQL Server.

04 — MODELAGEM

Transformação no Power BI

Consulta Power Query para processar itens de reprovação:

Power Query M
M LANGUAGE
let
    Fonte = CHECKLISTS,
    LinhasNaoVazias = Table.SelectRows(Fonte, 
        each ([ITENS_REPROVA] <> null and [ITENS_REPROVA] <> "")),
    ColunaListas = Table.AddColumn(LinhasNaoVazias, "ListaItens", 
        each Text.Split([ITENS_REPROVA], ";")),
    ExpandirItens = Table.ExpandListColumn(ColunaListas, "ListaItens"),
    ItensTrim = Table.TransformColumns(ExpandirItens, 
        {{"ListaItens", Text.Trim, type text}}),
    SelecionarItem = Table.SelectColumns(ItensTrim, {"ListaItens"}),
    ItensDistinct = Table.Distinct(SelecionarItem, {"ListaItens"}),
    ItensReprova = Table.RenameColumns(ItensDistinct, 
        {{"ListaItens", "ITEM"}})
in
    ItensReprova
